# Inaya Branding Store

A modern eCommerce web application built using Angular standalone architecture.

## 🚀 Features
- Product listing
- Add to cart
- Remove items from cart
- Quantity management
- Dynamic total price calculation
- Reactive state management using RxJS BehaviorSubject

## 🧠 Tech Stack
- Angular (Standalone Components)
- TypeScript
- RxJS
- HTML & CSS

## 📦 Architecture Highlights
- Component-based structure
- Shared state using services
- Reactive programming approach
- Clean and scalable code design

## ▶️ Run Locally
npm install
ng serve
